找到二叉树给定两结点的共同祖先

发表于2021-12-26|数据结构
|阅读量:

一、思路

如何得到一个给定结点的所有祖先？

我们很容易想到后序遍历，利用栈实现后序遍历时，遍历到某结点时，栈中所有元素即该结点祖先。

那么找出给定结点的所有祖先的算法如下：

//todo  输出x的所有祖先
void Anc_Print(BiTree T, TElemType x)
{
    BiTree p = T, r = NULL;    //r先用来标记右子树是否被访问过
    SqStack S; InitSqStack(S); //初始化栈
    while(p || S.top != -1)    //进行后序遍历
    {
        if(p)    //一直入栈直到树的左下
        {
            SqPush(S, p);
            p = p->lchild;
        }
        else    //  转为右子树
        {
            SqGetTop(S, p);
            if(p->rchild && p->rchild != r)    // 右子树存在且没被访问过
                p = p->rchild;    //转向
            else    //  右子树不存在或被访问过了
            {
                SqPop(S, p);
                if(p->data == x)
                    break;   //找到值为x的结点，退出循环
                r = p;
                p = NULL;
            }
        }
    }
    while(S.top != -1)    //输出所有祖先
    {
        SqPop(S, p);
        printf("%c  ",p->data);
    }
}

如何找到两结点共同点祖先？

由上面的过程，我们不难想到，后序遍历到某结点时，栈S中所有元素即该结点祖先。遍历到p或q时，将栈S中元素复制到一辅助栈S1里，再继续遍历到另一所求结点，那么从栈S栈顶开始逐个到辅助栈S1中去匹配，第一个匹配的结点即最近祖先。

二、实现

  1. 首先我们希望输入想要找的结点的值，就能通过此值定向到该结点（树中所有结点data域均不同），从而去寻找共同祖先，那么首先我们需要指向data域等于我们input值的指针，可以通过如下代码实现：

    //todo 找到指向x的指针
BiTree Find_x(BiTree T, TElemType x)
{
    if(T == NULL) return NULL;
    if(T->data == x)
        return T;
    if(Find_x(T->lchild, x))
        return Find_x(T->lchild, x);
    return  Find_x(T->rchild, x);
}

  2. 通过上述思想，将共同祖先保存到r：

    void Ancestor(BiTree T, BiTree p, BiTree q, BiTree &r)  //最后将最近共同祖先保存到 r
{
    BiTree temp = T, x = NULL;    //? temp为工作指针;先找到p,则x指向q,先找到q,则x指向p
    r = NULL;    //? r先用来标记右子树是否被访问过
    int i = 0, j = 0, tag = 0;    //?  tag记录是否找到了p或q(此时栈S1中已复制完成)
    SqStack S; InitSqStack(S);
    SqStack S1; InitSqStack(S1);
    while(temp || S.top != -1)
    {
        if(temp)    //? 一直入栈直到树的左下
        {
            SqPush(S, temp);
            temp = temp->lchild;
        }
        else    //?  转为右子树
        {
            SqGetTop(S, temp);
            if(temp->rchild && temp->rchild != r) //? 右子树存在且没被访问过
                temp = temp->rchild;    //转向
            else    //?  右子树不存在或被访问过了
            {
                SqPop(S, temp);        //* 将栈顶结点出栈，让 temp指向它
                if(!tag && (temp == p || temp == q))    //*  tag = 1时，说明已找到p或者q,所以此if下语句全程只会执行一次
                {
                    x = temp==p ? q : p;
                    for(i = 0; i <= S.top; i++)
                        S1.data[i] = S.data[i];
                    S1.top = S.top;
                    tag = 1;
                }
                if(temp == x)
                    break;
                r = temp;
                temp = NULL;
            }
        }
    }
    int i0 = i;  //i0存放S1的栈顶
    j = S.top;
    while(j >= 0)    //栈S从栈顶开始，从辅助栈S1中逐个寻找相同元素
    {
        while(i>=0 && S.data[j] != S1.data[i])
            i--;
        if(i < 0)
        {
            i = i0;  //i重新指向栈顶
            j--;  //继续从栈S下一个元素开始逐个与S1匹配
        }
        else  //i>=0说明匹配成功，退出循环
            break;
    }
    if(i >= 0)
        r = S1.data[i];
    else  //如果i<0，则寻找失败，无共同祖先
        r = NULL;
}

  3. 返回指向r的指针（考虑到上面算法会出错的情况并完善）：

    BiTree ComAnc_Node(BiTree T, TElemType p, TElemType q)
{
    BiTree r, n1, n2;
    n1 = Find_x(T, p);
    n2 = Find_x(T, q);
    if(!n1 || !n2)
        return NULL;
    if(p==T->data || q==T->data)
        return T;
    Ancestor(T, n1, n2, r);
    return r;
}
